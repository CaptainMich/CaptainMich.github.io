Label and Button

In line 3 we imported the built-in tkinter module; in line 6 we assigned an instance of the Tk class, by calling its constructor, to the variable 'win'. In the next line we gave a title to our window

if we want to prevent the window from being resized we can add the commented code line win.resizable(0, 0); the maximize button in the toolbar of the window should be grayed out. Note that if we pass to the resizable() method other values, we hard-code the x and y start up size of the GUI, but that won't make it nonresizable.

In lines 11-12 we added the label to the GUI by passing our window instance into the ttk.Label constructor and setting the text property. This becomes the text our Label will display. We have also made use of the grid layout manager to set the position into the window.

In lines 15 to 17 we defined the event handler that is being invoked once the button gets clicked; this function just change the label foreground color and the text inside the button. Then in lines 19-20 we created the button and binded the command to the clickMe() function.

GUIs are event-driven. Clicking the button creates an event. We binded what happens when this event occurs in the callback function using the command property of the ttk.Button widget

By adding the property state='disabled' we can also disable widgets (line 25)

In the last line we started the window's event loop by calling the mainloop() method on the class instance 'win'. Up to this point in our code, the GUI will not be displayed until we start the main event loop.

from tkinter import ttk

win = tk.Tk()                                                       # create an istance of Tk class  by calling its constructor
win.title("Label and Button")                                       # give our window a title via the title property

label =ttk.Label(win, text="A Label")                               # assign label to a variable
label.grid(column=0, row=0)                                         # position the label within the form
