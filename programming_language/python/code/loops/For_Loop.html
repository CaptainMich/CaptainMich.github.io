For Loop

B elow we cover all the possibility to make a for loop:

  • for loop : is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string)
  • break : is used to stop the loop before it has looped through all the items
  • continue : is used to stop the current iteration of the loop, and continue with the next
  • range() function : it returns a sequence of numbers, starting from x (0 by default), and increments by y (1 by default), and ends at a specified number.

For Loop 


# -------------------------------------------------------------------------------------------------
# FOR LOOP
# -------------------------------------------------------------------------------------------------


foods = ["Bacon" , "Tuna" , "Ham" , "Sausages" , "Beef"]      # for loop (1)
for i in foods:                                               # ...
    print(i)                                                  # ...
    print("Len is:", len(i))                                  # ...

print ('')

foods = ["Bacon" , "Tuna" , "Ham" , "Sausages" , "Beef"]      # for loop (2)
for i in foods[:2]:                                           # ...
    print(i)                                                  # ...

adj = ["red", "big", "tasty"]                                 # for loop (3)
fruits = ["apple", "banana", "cherry"]                        # ...
for x in adj:                                                 # ...
  for y in fruits:                                            # ...
    print(x, y)                                               # ...

Break and Continue 


# -------------------------------------------------------------------------------------------------
# BREAK and CONTINUE
# -------------------------------------------------------------------------------------------------

magicNumber = 42
                                                        # example break statement
for n in range(101):                                    # ...
  if n is magicNumber:                                  # ...
    print(n, "is the magic number!")                    # ...
    break                                               # ...
  else:                                                 # ...
    print(n)                                            # ...
                                                        # ...
print('')                                               # ...

numbersTaken = [2, 5, 12, 33, 17]                       # example continue statement
                                                        # ...
print('Here are the numbers that are taken:')           # ...
print(numbersTaken)                                     # ...
print('')                                               # ...
                                                        # ...
print('Here are the numbers that are still avaiable:')  # ...
for n in range(1, 20):                                  # ...
  if n in numbersTaken:                                 # ...
    continue                                            # ...
  print(n)                                              # ...
                                                        # ...

Range() function 


# -------------------------------------------------------------------------------------------------
# RANGE
# -------------------------------------------------------------------------------------------------

for x in range(10):                           # to make an action ('n') times
  print("Pippo")                              # ...
print('')                                     # ...

for x in range(5, 12):                        # second example -->  (from 'x' to 'y')
  print(x)                                    # ...
print('')                                     # ...

for x in range(10, 40, 5):                    # third example -->  (from 'x'[10] to
  print(x)                                    # 'y'[40] in steps of 'z'[5])
print('')                                     # ...

print(list(range(4)))                         # create new list from 0 to 3
print(list(range(-6,7,2)))                    # -6 to 7 by 2

print([[x ** 2, x ** 3] for x in range(4)])                 # some more complex example
print([[x, x / 2, x*2] for x in range(-6, 7, 2) if x > 0])  # ...
