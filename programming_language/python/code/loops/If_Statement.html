Assert Statement
Assert : is generally used during debugging code session. It lets you test if a condition in your code returns True, if not, the program will raise an AssertionError 

# -------------------------------------------------------------------------------------------------
# ASSERT
# -------------------------------------------------------------------------------------------------

x = int(input("Enter a number greater than 10: "))      # example assert statement
assert x > 10, "Invalid input"                          # ...

print("Entered value is: {}\n".format(x))               # ...

        score += x
