Scope of Variables

Not all variables are accessible from all parts of our program, and not all variables exist for the same amount of time. Where a variable is accessible and how long it exists depend on how it is defined. We call the part of a program where a variable is accessible its scope, and the duration for which the variable exists its lifetime.

  • global variable: a variable which is defined in the main body of a file; it will be visible throughout the file, and also inside any file which imports that file. Global variables can have unintended consequences because of their wide-ranging effects that is why we should almost never use them
  • local variable : a variable which is defined inside a function; it is accessible from the point at which it is defined until the end of the function, and exists for as long as the function is executing. 


# -------------------------------------------------------------------------------------------------

# -------------------------------------------------------------------------------------------------

 print("z local is: {}".format(z))                          # access local variable
 print("z global is: {}".format(globals()['z']))            # access local variable

Return Values

After defining what a "normal" function is, it's noticeable that a function can return a value. So we can define a function which will return a value back, we will store the returned value in a variable and we will print it. 



# -------------------------------------------------------------------------------------------------
# RETURN VALUE
# -------------------------------------------------------------------------------------------------

def sample(number):                                  # define the function
    operation = (number / 2 + 7)                     # body of the function
    return operation                                 # return a value


test_sample = sample(20)                             # storage the returned value
print('Testing Sample: ' + str(test_sample) + '\n')  # print it

def calc(a,b):                                       # define the function
    add = a + b                                      # ...
    diff = a - b                                     # ...
    mul = a * b                                      # ...
    div = a / b                                      # ...

    return add, diff, mul, div                       # return multiple values

result = calc(8,4)                                   # ...
print(result)                                        # ...
Default Values for Arguments

A function can be defined with default value: it's a value that will be returned everytime the function is called without passing any argument 



# -------------------------------------------------------------------------------------------------
# DEFAULT VALUES for ARGUMENTS
# -------------------------------------------------------------------------------------------------

def get_gender(sex = 'Unknown'):    # define a function with a DEFAULT VALUE
    if sex is 'm':                  # body of the function (if - elif condition)
        sex = 'Male'                # ...
    elif sex is 'f':                # ...
        sex = 'Female'              # ...
    print(sex)                      # body of the function (print the final sex)

get_gender('m')                     # call the function
get_gender('f')                     # many time to display
get_gender()                        # the functionality

print('')
Keyword Arguments

It's possible to define function with multiple default values associated with a keyword. As we can see, we can replace every default value simply by passing the keyword and the value; if we don't associate the keyword, the values will be replaced in order 



# -------------------------------------------------------------------------------------------------
# KEYWORD ARGUMENTS
# -------------------------------------------------------------------------------------------------

# define a function with 3 default value
def dumb_sentence(name = 'Pippo', action = 'ate', item = 'nuts'):
    print(name,action,item)

dumb_sentence()                                       # call the function
dumb_sentence(name = 'SuperPippo')                    # use one different keyword arg
dumb_sentence(action = 'drive', item = 'a car')       # use two different keyword args
dumb_sentence('Captain','plays','guitar')             # use multi different keywords args

Unpacking Arguments

We have two different ways to pass arguments from a list to a function: the first one is pass the arguments one by one; the other one is pass a packed argument list as function argument and unpack it (it should be noted that the list must contain the same number of arguments as the number of arguments taken by the function ) 



# -------------------------------------------------------------------------------------------------
# UNPACKING ARGUMENTS
# -------------------------------------------------------------------------------------------------

def healt_calculator(age, apples_ate, cigs_smoked):
    answer = (100 - age) + (apples_ate * 0.35) - (cigs_smoked * 1)
    print(answer)

data = [20, 2, 0]
                                                   # call the function and ...
healt_calculator(data[0],data[1],data[2])          # ...pass arguments one by one ...
healt_calculator(*data)                            # ...or unpacking argument list

print('')
Recursive Function

A recursive function is a function that calls itself


# -----------------------------------------------------------------------------------------------
# RECURSIVE FUNCTION
# -----------------------------------------------------------------------------------------------

def factorial(n):                              # define a function
 if (n == 0):                                  # ...
   return 1                                    # ...
 else:                                         # ...
   return n * factorial(n-1)                   # call the function recursive

result = factorial(3)                           # ...
print(result)                                   # ...
Lambda Function

A lambda function is a small anonymous function which can take any number of arguments, but can only have one expression.


# -------------------------------------------------------------------------------------------------
# LAMBDA
# -------------------------------------------------------------------------------------------------

def power(n):                             # lambda inside function
  return lambda a : a * n                 # ...

result = power(2)                         # ...
print("Function: {}".format(result(3)))   # ...

f = lambda n:n**3                         # using lambda
print("Lambda: {}".format(f(2)) )         # ...

# example using lambda: even or odd
l = lambda x: 'YES' if x % 2 == 0 else 'NO'
print("Number is Even? {}".format(l(10)))
print("Number is Even? {}".format(l(9)))

# example using lambda: sum of numbers
s = lambda a,b: a + b
print("Sum is {}".format(s(10,5)))
Decorator

Decorators are very powerful and useful tool in Python since it allows programmers to modify the behavior of function or class. Decorators allow to wrap another function in order to extend the behavior of wrapped function, without permanently modifying it.
In Decorators, functions are taken as the argument into another function and then called inside the wrapper function.


# -------------------------------------------------------------------------------------------------
# DECORATOR
# -------------------------------------------------------------------------------------------------

def decorator(func):                                # defining a decorator
  def inner():                                      # inner is a Wrapper function in
    print("This is before function execution")      # which the argument is called
    func()                                          # inner function can access the outer local
    print("This is after function execution")       # functions like in this case "func"
  return inner                                      # calling the actual function now
                                                    # inside the wrapper function.
@decorator
def function_to_be_used():                          # defining a function, to be called inside wrapper
    print("This is inside the function !!")         # ...

# passing 'function_to_be_used' inside the
# decorator to control its behavior

# calling the function
function_to_be_used()
