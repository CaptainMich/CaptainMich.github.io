Web Type

A Project by Captain Mich

August 13, 2018

Web Type is a simple project for semantic and readable typography to use on the web. You should Fork it on Github.

Download ZIP Download TAR Fork on Github

Why It's Awesome

In case you still aren't convinced, here's why you should give Web Type a try:

  • Built with Sass
  • Uses semantic HTML5 elements and ARIA Roles.
  • Lightweight and easy to modify
  • It's responsive
